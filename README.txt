-- 專案：TheGameC | 遊戲購買平台 --

個人作品
主題：模擬用戶購買商品的平台
架構：首頁、商店、商品、購物車、客服、關於我們、個人資訊、登入頁
功能：購物車、登入/註冊、第三方登入(Google)、修改密碼(Email驗證)、客服表單、搜索功能、留言/評論功能
開發方式：
        後端：Express (Node.js)
        前端：React
        資料庫：MySQL
